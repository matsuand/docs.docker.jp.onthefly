読む時間の目安: 106 分

本ページでは Docker Desktop 最新版（Edge）に関する情報を示します。 最新版では、最新の機能を試すことができます。 ただし一部の機能は試験的（experimental）なものであり、安定版（Stable）には導入されないかもしれません。

安定版については 安定版リリースノート を参照してください。 また Docker Desktop のシステム要件については インストール前に確認すべきこと を参照してください。
