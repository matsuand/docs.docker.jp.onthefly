docker trust signer remove

読む時間の目安: 3 分

説明

署名者（signer）を削除します。

利用方法

$ docker trust signer remove [オプション] NAME REPOSITORY [REPOSITORY...]

追加説明

docker trust signer removeは署名されたリポジトリから署名者（signer）を削除します。

本コマンドの利用例については、以下に示す 利用例の節 を参照してください。

オプション

名前／省略形 デフォルト 説明
--force , -f 最新の署名者を削除する際に確認プロンプトを表示しません。

利用例

リポジトリからの署名者削除

以下のようなリポジトリから、追加済みの署名者aliceを削除していきます。

$ docker trust inspect --pretty example/trust-demo

No signatures for example/trust-demo


List of signers and their keys:

SIGNER              KEYS
alice               05e87edcaecb
bob                 5600f5ab76a2

Administrative keys for example/trust-demo:
Repository Key: ecc457614c9fc399da523a5f4e24fe306a0a6ee1cc79a10e4555b3c6ab02f71e
Root Key:       3cb2228f6561e58f46dbc4cda4fcaff9d5ef22e865a94636f82450d1d2234949

docker trust signer removeを実行してaliceを削除します。

$ docker trust signer remove alice example/trust-demo

Removing signer "alice" from image example/trust-demo...
Enter passphrase for repository key with ID 642692c:
Successfully removed alice from example/trust-demo

docker trust inspect --prettyの実行により、今度はaliceが署名者として表示されなくなりました。

$ docker trust inspect --pretty example/trust-demo

No signatures for example/trust-demo


List of signers and their keys:

SIGNER              KEYS
bob                 5600f5ab76a2

docker trust signer removeはリポジトリからの署名者の削除にあたっては、できる限り削除を行っていきます。 つまり 1 つの署名者削除に失敗しても、続くリポジトリからの削除処理は続行します。

$ docker trust signer remove alice example/unauthorized example/authorized

Removing signer "alice" from image example/unauthorized...
No signer alice for image example/unauthorized

Removing signer "alice" from image example/authorized...
Enter passphrase for repository key with ID c6772a0:
Successfully removed alice from example/authorized

Error removing signer from: example/unauthorized

上位コマンド

コマンド 説明
docker trust signer Docker イメージに対して署名可能なエンティティを管理します。

関連コマンド

コマンド 説明
docker trust signer add 署名者（signer）を追加します。
docker trust signer remove 署名者（signer）を削除します。
