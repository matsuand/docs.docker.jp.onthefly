docker trust sign

読む時間の目安: 2 分

説明

イメージに署名します。

利用方法

$ docker trust sign IMAGE:TAG

追加説明

docker trust signは署名されたリポジトリを生成するために、タグへの署名追加を行います。

本コマンドの利用例については、以下に示す 利用例の節 を参照してください。

オプション

名前／省略形 デフォルト 説明
--local タグづけされているイメージをローカルにおいて署名します。

上位コマンド

コマンド 説明
docker trust イメージ上の Docker トラストを管理します。

関連コマンド

コマンド 説明
docker trust inspect 鍵と署名に関する低レベル情報を返します。
docker trust key 署名している Docker イメージの鍵を管理します。
docker trust revoke イメージのトラスト情報を削除します。
docker trust sign イメージに署名します。
docker trust signer Docker イメージに対して署名可能なエンティティを管理します。
