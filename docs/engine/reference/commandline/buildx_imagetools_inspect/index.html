docker buildx imagetools inspect

読む時間の目安: 18 分

説明

レジストリ内イメージの詳細を表示します。

利用方法

$ docker buildx imagetools inspect [オプション] NAME

オプション

名前／省略形 デフォルト 説明
--format {{.Manifest}} Format the output using the given Go template
--raw オリジナルの未フォーマットの JSON マニフェストを表示します。
--builder 設定された builder インスタンスをオーバーライドします。

利用例

Override the configured builder instance (--builder)

Same as buildx --builder.

Format the output (--format)

Format the output using the given Go template. Defaults to {{.Manifest}} if unset. Following fields are available:

  • .Name: provides the reference of the image
  • .Manifest: provides the manifest or manifest list
  • .Image: provides the image config
  • .BuildInfo: provides build info from image config

.Name

$ docker buildx imagetools inspect alpine --format "{{.Name}}"
Name: docker.io/library/alpine:latest
