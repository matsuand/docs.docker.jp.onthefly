docker-compose restart

  • 英語表記
  • 日本語訳
Usage: restart [options] [SERVICE...]

Options:
  -t, --timeout TIMEOUT      Specify a shutdown timeout in seconds.
                             (default: 10)

利用方法: restart [オプション] [SERVICE...]

オプション:
  -t, --timeout TIMEOUT      シャットダウンに要するタイムアウト時間を秒数で指定します。
                             （デフォルト: 10）

停止中および実行中のサービスを再起動します。

docker-compose.ymlの設定を変更していたとしても、このコマンド実行後にその変更内容は反映されません。

たとえば環境変数への設定変更（コンテナーのビルド後に追加した場合や、コンテナーコマンドが実行される前である場合）は、再起動を行っても設定更新されません。

サービスの再起動ポリシーに関する情報については、Compose ファイル v3 の restart や Compose v2 の restart を参照してください。 なお Swarm モードでのスタックのデプロイ を行う場合には、それとは違って restart_policy を用いることが必要です。
